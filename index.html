Frequently Asked Questions

Everything you need to know about DeployBar

What is DeployBar?

DeployBar is a free, open-source macOS menu bar app that lets you monitor your Vercel deployments and GitHub Actions workflows in real-time. Instead of constantly switching tabs to check deployment status, you get instant notifications and updates right from your menu bar.

Is DeployBar free?

Yes! DeployBar is completely free and open-source. You can download it, use it, and even contribute to its development on GitHub.

Which macOS versions are supported?

DeployBar works on macOS 11 (Big Sur) and later. It supports both Apple Silicon (M1/M2/M3) and Intel-based Macs.

How do I get started?
  1. Download DeployBar for your Mac (Apple Silicon or Intel)
  2. Generate a Vercel token at vercel.com/account/tokens
  3. Generate a GitHub token with 'repo' and 'workflow' scopes
  4. Add both tokens in DeployBar settings
  5. Your deployments will start appearing in the menu bar automatically
How often does DeployBar refresh?

DeployBar automatically refreshes your deployments every 30 seconds. You can also manually refresh anytime by clicking the refresh icon in the menu bar.

Are my tokens stored securely?

Absolutely! Your Vercel and GitHub tokens are stored in the macOS system keychain, not in plain text or localStorage. This is the same secure storage that Apple uses for passwords and sensitive data.

Can I monitor private repositories and deployments?

Yes! DeployBar works with both public and private repositories. As long as your GitHub token has the correct permissions, you can monitor all your private deployments and workflows.

Does DeployBar support multiple accounts?

Yes! You can switch between personal and team accounts effortlessly. Perfect for developers working across multiple organizations.

Will I get notifications for failed deployments?

Yes! DeployBar sends native macOS notifications when deployments fail, so you can catch issues immediately before your users do.

Can I see detailed information about deployments and workflows?

Absolutely! Click any deployment or workflow run to open a detailed view. For Vercel deployments, you'll see build logs, commit information, error details, and more. For GitHub workflows, you'll see jobs, steps, commit details, and real-time updates as the workflow progresses. All detail pages update automatically while builds are running.

Why isn't the app opening after download?

Since DeployBar isn't notarized by Apple (to avoid the $99/year fee), macOS blocks it by default. Run this command in Terminal:

$ xattr -d com.apple.quarantine /Applications/DeployBar.app

Then open DeployBar from Applications. This is a one-time setup.

How do I uninstall DeployBar?

Simply drag DeployBar.app from Applications to Trash. Your tokens will be removed from the keychain automatically.

Can I contribute to DeployBar?

Absolutely! DeployBar is open-source. Visit github.com/waseemmaya/deploy-bar to report issues, suggest features, or submit pull requests.

Setup

Up and running in 2 minutes

01

Install DeployBar

Option A: Install via Homebrew (recommended):

$ brew tap waseemmaya/deploy-bar-updates https://github.com/waseemmaya/deploy-bar-updates
$ brew install --cask deploybar

Option B: Manual download — grab the DMG from below and drag to Applications. Since the app isn't notarized by Apple (due to the $99/year fee), run this command in Terminal before opening:

$ xattr -d com.apple.quarantine /Applications/DeployBar.app

If the command fails, try with sudo at the beginning. Then open DeployBar from Applications.

02

Generate a Vercel Token

Head to vercel.com/account/tokens and create a new access token. Give it a descriptive name like "DeployBar".

03

Generate a GitHub Token

Go to GitHub Token Settings and create a personal access token with repo and workflow scopes to monitor your Actions.

04

Connect Your Accounts

Add your Vercel and GitHub tokens in DeployBar settings. They're securely stored in your system keychain — you'll never need to enter them again.

05

Monitor Everything

That's it! Your Vercel deployments and GitHub Actions will appear in the menu bar. Click any item to see detailed information including build logs, commit details, jobs, and steps. Or click the external link icon to open it in your browser.
